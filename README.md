# ai_uni

A collection of CSV datasets and code files used in the **AI Basics** university course.

## Contents

- **CSV files** – datasets used for AI/ML exercises and experiments
- **Code files** – scripts and notebooks that load and process the CSV data

## Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/s3drik/ai_uni.git
   ```
2. Open the relevant code file for a lesson.
3. Make sure the CSV file referenced in the code is in the same directory (or update the path).
4. Run the script to reproduce the results.
